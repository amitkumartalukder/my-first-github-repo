CSS Box Model হলো এমন একটি ধারণা, যা নির্ধারণ করে একটি HTML element ওয়েব পেজে কতটুকু জায়গা দখল করবে। একটি element মোট কতটুকু জায়গা নেবে, তা নির্ভর করে নিচের বিষয়গুলোর উপর— Content: Content-এর width ও height যত বড় হবে, element তত বেশি জায়গা নেবে। Padding: Content-এর চারপাশে ভেতরের ফাঁকা জায়গা। Padding বাড়ালে element-এর মোট আকারও বাড়ে। Border: Content-এর চারপাশের বর্ডার। Border মোটা হলে element আরও বেশি জায়গা নেয়। Margin: Element-এর বাইরের ফাঁকা জায়গা। Margin বাড়ালে অন্য element-এর সাথে দূরত্ব বাড়ে এবং মোট জায়গাও বেশি লাগে।

CSS Box Modeldepends on the 4 property:

  1. Content
  2. Border
  3. Margin
  4. Padding
