Today we can learn about css

CSS: Cascading Style Sheet. Basically it is used for adding Style in html structure.We can use 3 types of css:

  1. Inline
  2. Internal or Embeded
  3. External

Css has 3 part:

  1. Selector
    • Selector can be used to select the element what we want to select.
  2. Property
    • what type of property we use like color etc.
  3. Value
    • what type of color / etc
