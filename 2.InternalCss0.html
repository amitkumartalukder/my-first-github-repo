In this section we learn about internal css

Inline CSS-এর অসুবিধা হলো প্রতিটি element-এ আলাদাভাবে বারবার CSS লিখতে হয়। ফলে কোডের লাইন বেড়ে যায় এবং maintain করা কঠিন হয়ে পড়ে। তাই একাধিক element-এ একই style প্রয়োগ করার ক্ষেত্রে আমরা Internal CSS ব্যবহার করি।
